# The Uplink

**Purpose**: To make it easier on a call with transatlantic clients to find common ground.

## Overview

The Uplink is a dashboard utility designed to bridge the gap between distributed teams. It provides real-time context about your clients' environments, helping to build rapport and streamline communication across time zones.

## Features

- **Timezone Grid**: Real-time local time, date, and weather for key locations (LAX, NYC, WRO, LVI).
- **Silence Killer V2**: Generates random icebreakers and trivia to kickstart conversations.
- **Overlap Visualizer**: A Gantt-style chart to visualize working hour overlaps between time zones.
- **Converter**: Quick conversion tools for Temperature (C/F), Weight (Lbs/Kg), and Length (Imperial/Metric).
- **Upcoming Holidays**: Tracks upcoming holidays in relevant regions to aid in scheduling.
- **Business Clock**: Visualizes the current quarter's progress to align on business timelines.
